FlashCards sobre programación web

¿Qué es un lenguaje de programación?

Es un lenguaje formal que, mediante una serie de instrucciones, le permite a un programador escribir un conjunto de órdenes, acciones consecutivas, datos y algoritmos para, de esa forma, crear programas que controlen el comportamiento físico y lógico de una máquina.

¿Qué herramientas son usadas para crear una pagina web?

Ventajas de HTML5

  • Posibilita la inserción de vídeos y audio de forma directa.
  • Es nativo, y por tanto independiente de plugins de terceros
  • Es más semántico, con etiquetas que permiten clasificar y ordenar en distintos niveles y estructuras el contenido

¿Qué es un servidor Web?

Un servidor web es la computadora en la que se almacenan los archivos de las páginas web.

¿Qué es DHTML?

El lenguaje de marcado de hipertexto dinámico permite una mayor interactividad y flexibilidad de diseño de página porque cada elemento es un objeto y se puede acceder a cada objeto y ajustarlo por separado.

¿Cuál es la diferencia entre Java Script y Java?

Java Script es un código que agregas a tu código XHTML, el navegador lee el código y lo ejecuta. Cualquiera puede leer código Java Script. Java es un lenguaje compilado. Antes de que pueda ejecutarse, debe ser procesado por un programa especial para crear un archivo.

HTTP

Protocolo de Transferencia de Hipertexto. Es el protocolo que utilizan los navegadores y los servidores web para comunicarse. Se basa en el protocolo de menor nivel TCP/IP.

DNS

son los servidores y el nombre del protocolo que se encarga de traducir nombres de dominio de páginas Web en direcciones IP de los servidores que las sirven.

CDN

es una red de distribución de contenidos o Content Delivery Network. Cuando un usuario solicita una página o cualquier otro recurso a un servidor web

Desventajas de HTML5

  • No hay un formato de vídeo compatible con todos los navegadores web
  • lleva tiempo tener listas las aplicaciones móviles
  • Produjo discrepancias en la implementación de atributos CSS

CSS

CSS es un lenguaje para la composición y estructuración de páginas web (HTML o XML). Este lenguaje contiene elementos de codificación y se compone de estas “Cascading Style Sheets” que también se llaman archivos CSS (. css).

PHP

PHP es un lenguaje de programación de uso general que se adapta especialmente al desarrollo web.​

GitHub

GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones Git. Se utiliza principalmente para la creación de código fuente de programas de ordenador.​

Dominio

Un dominio de internet es un nombre único que identifica a una subárea de internet. El propósito principal de los nombres de dominio es traducir las direcciones IP de cada activo en la red, a términos memorizables y fáciles de encontrar. ​

¿Qué es un repositorio?

Un repositorio es un espacio centralizado donde se almacena, organiza, mantiene y difunde información digital, habitualmente archivos informáticos, que pueden contener trabajos científicos, conjuntos de datos o software. ​

Neocities

Neocities es un servicio de alojamiento web comercial para páginas estáticas. ​

Figma

Figma es un editor de gráficos vectorial y una herramienta de generación de prototipos, principalmente basada en la web, con características off-line adicionales habilitadas por aplicaciones de escritorio en macOS y Windows. ​

Micrositio

Un micrositio, también conocido como minisite o weblet, es un término de diseño web que se refiere a una página web individual o a un grupo de páginas que extienden o amplían la funcionalidad de sitios web primarios.​

FrontEnd

El desarrollo web Front-end consiste en la conversión de datos en una interfaz gráfica para que el usuario pueda ver e interactuar con la información de forma digital usando HTML, CSS y JavaScript. ​

¿Qué es una página web?

Una página web​​ es un documento o información electrónica capaz de contener texto, sonido, vídeo, programas, enlaces, imágenes, hipervínculos y muchas otras cosas, adaptada para la llamada World Wide Web, y que puede ser accedida mediante un navegador web. ​
